메쉬코리아 / QA팀 / 2019.01.02 ~ 재직중
1. VROONG 도메인 업무
1) 개요
- VROONG 상점주들을 위한 POS PC/M 서비스
- POS PC/M 전체 기능 정합성 확인
- 상점주 Direct 가입 페이지 신규 기능 정합성 확인
- Admin 사이트 개선 및 신규 기능 정합성 확인
- 네이버 톡톡 OMS 연동 신규 기능 정합성 확인
- 네이버 톡톡 OMS 출시
- VROONG Direct 출시
2) 대상
- 폐점상점 POS 로그인 및 UI 변경 QA 진행 (2019.01)
- 원클릭 전화번호 입력 주문 접수 QA 진행 (2019.01)
- 상점 개점/폐점 예약 Admin 페이지 QA 진행 (2019.02)
- POS PC/M 예치금 마이너스 방지 QA 진행 (20190.2)
- POS PC OMS 주문서 출력 추가 기능 QA 진행 (2019.02)
- POS M 원클릭 퍼미션 대응 QA 진행 (2019.03)
- 네이버 OMS 구축 QA 진행 (2019.03 ~2019. 05)
- VROONG Direct Web QA 진행 (2019.06)
- 상점 긴급공지 QA 진행 (2019.07)
- 상점 취소 수수료 기사 지급 QA 진행 (2019.08)
- Admin 페이지 주소 변경 시 할증 업데이트 정산 QA 진행 (2019.09)
- Admin 재이동 배송비 추가 정산 QA 진행 (2019.10)
3) 담당업무
- 사용자 유스케이스 요구사항 분석
- 테스트케이스 설계 및 작성
- 테스트 계획 수립 및 테스트 인력 MM관리
- QA 프로젝트 인력대상 테스트 리딩 및 점검수행
- POS, 기사, 매니저, Admin 가입/조회/변경 등 상호 연동 확인
- Admin Web 설정 및 POS/기사/매니저 연동 노출 확인
- Web 이슈 발생 시 Console, Network log 확인
- 이슈 처리 완료 시 결함 수정된 기능에 대한 Side Effect 확인
- 이슈 추적관리 및 통합보고
- 테스트 결과서 작성 및 보고
- PO, 개발자와 오류이슈 협업
4) 배포관리
- 상용 적용 후 안정성 검토
- B2C 보고 이슈 대응 및 검증 사항 보고
